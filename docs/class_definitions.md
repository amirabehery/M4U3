# Class Definitions — Construction Site Safety Dataset

Dataset: Construction Site Safety Image Dataset (Roboflow export via Kaggle)

Classes (10):

1. Hardhat — worker wearing safety helmet
2. Mask — worker wearing face mask
3. NO-Hardhat — worker head without helmet
4. NO-Mask — worker without face mask
5. NO-Safety Vest — worker without safety vest
6. Person — human detected on site
7. Safety Cone — traffic / warning cone
8. Safety Vest — high-visibility vest
9. machinery — construction equipment / machines
10. vehicle — trucks, cars, construction vehicles

Label Notes:
- Bounding boxes surround visible object area
- Partial visibility allowed
- Overlapping objects labeled separately
- PPE violations labeled using NO-* classes
