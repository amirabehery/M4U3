# M4U3 — Construction Site Safety Detection (YOLOv8)

Object detection model for construction site safety to detect:
helmets, masks, safety vests, PPE violations, workers, cones, machinery, and vehicles.

## Dataset

Construction Site Safety Image Dataset (Roboflow export via Kaggle)

Source:
https://www.kaggle.com/datasets/snehilsanyal/construction-site-safety-image-dataset-roboflow

Classes: see docs/class_definitions.md

## Repository Structure

- notebooks/ — training and inference notebooks (Colab-ready)
- docs/ — class definitions, error analysis, governance checklist
- results/ — curves and prediction evidence
- weights/ — trained model weights link or files
